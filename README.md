# Power BI Carnival Data Project

This repository contains the Power BI project developed to analyze and visualize Carnival-related data using Excel datasets from 2022 to 2025.  
The dashboards provide insights into street blocks, samba schools, geographic distribution, operational metrics, and year-over-year trends.
